### 基本数据结构


| **时间复杂度** | **阶**             | **f(n) 举例** |
| -------------- | ------------------ | ------------- |
| 常数复杂度     | O(1)               | 1             |
| 对数复杂度     | O(logn)            | logn + 1      |
| 线性复杂度     | O(n)               | n + 1         |
| 线性对数复杂度 | O(nlogn)           | nlogn + 1     |
| k 次复杂度     | O(n²)、O(n³)、.... | n² + n +1     |
| 指数复杂度     | O(2n)              | 2n + 1        |
| 阶乘复杂度     | O(n!)              | n! + 1        |
